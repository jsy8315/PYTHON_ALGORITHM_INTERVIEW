# 자료형

## 001. 파이썬 자료형

### 숫자
* 파이썬에서는 숫자 정수형으로 int만을 제공
* bool은 논리 자료형, 1(True)와 0(False)으로 처리되는 int의 서브 클래스
* object > int > bool

### 매핑
* 키와 자료형으로 구성된 복합 자료형
* 파이썬은 딕셔너리

### 집합
* set은 중복된 값을 갖지 않는 자료형
* 입력 순서가 유지되지 않는다
* ex.

        a = {3, 2, 3, 5}
        a 출력시
        {2, 3, 5}


### 시퀀스
* 어떤 특정 대상의 순서 있는 나열을 뜻함
* 불변(str, tuple, bytes),  가변(list 등)이 존재
